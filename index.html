New Released Lego Sets Wishlist

The Barad-dûr set is a highly anticipated LEGO set that brings the iconic tower from the Lord of the Rings universe to life. This detailed set includes intricate architecture and is a must-have for any fan of the series. Standing tall, the set showcases the dark and foreboding essence of Sauron's stronghold, perfect for display or imaginative play. This set is retailed at $500 usd. I'd say if you have the money for it, BUY IT!
