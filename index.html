New Released Lego Sets Wishlist

The Barad-dûr set is a highly anticipated LEGO set that brings the iconic tower from the Lord of the Rings universe to life. This detailed set includes intricate architecture and is a must for any fan of the series. Standing tall, the set showcases the dark and foreboding essences of Sauron's stronghold, perfect for display or imaginative play. This set is retailed at $459.99 usd. I'd say if you have the money for it, BUY IT!

Jabba's Sail Barge is another LEGO set from the Star Wars universe. It recreates the famous scene from "Return of the Jedi," where Luke Skywalker faces off against Jabba the Hutt. The set including details of the sail barge, along with multiple minifigures like Jabba, Leia, and R2-D2. It's perfect for collectors and Star Wars fans who want to bring this legendary scene to life. This set is retailed at $500.00, so if you save up a little bit of money, its yours!
